# .NET 10 Basit Web Sitesi

Bu proje, `net10.0` hedefleyen sade bir ASP.NET Core web uygulamasıdır.

## Çalıştırma

```bash
dotnet restore
dotnet build
dotnet run
```

Tarayıcıda aç:

- Ana sayfa: `http://localhost:5000` veya `https://localhost:5001`
- API: `http://localhost:5000/api/message`

## Not

Makinede .NET 10 SDK kurulu olmalı.
